# Z80_TEST_
У цьому пості я вам покажу простий прототип компютера на базі Z80 


Освітній 8-бітний комп’ютер на базі процесора Z80.  
Це ранній прототип, який перебуває на стадії розробки.  
Апаратна частина, схема, плата та прошивка ще експериментальні,  
тому немає гарантії, що ця ревізія повністю працездатна.

Мета проєкту — спроєктувати та створити власну систему на Z80  
із зовнішньою RAM/ROM, дешифрацією адрес, підсистемою вводу/виводу  
та мінімальною прошивкою. Усі файли надаються для навчання та досліджень.



Educational 8-bit computer based on the Z80 CPU.  
This is an early prototype currently in development.  
The hardware, PCB and firmware are still experimental,  
so there is no guarantee that this revision is fully functional.

The goal of the project is to design and build a Z80-based system  
with external RAM/ROM, address decoding, an I/O subsystem  
and minimal firmware. All files are provided for educational purposes.
